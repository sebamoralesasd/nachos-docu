Detailed Description

template<class Item>
class ListElement< Item >

Data structures to manage LISP-like lists.

As in LISP, a list can contain any type of data structure as an item on the list: thread control blocks, pending interrupts, etc.

The following class defines a “list element” – which is used to keep track of one item on a list.

Internal data structures kept public so that List operations can access them directly.
