◆ VoidFunctionPtr

typedef void(* VoidFunctionPtr) (void *arg)

This declares the type VoidFunctionPtr to be a “pointer to a function taking a pointer argument and returning nothing”.

With such a function pointer (say it is "func"), we can call it like this: func (arg); or: (*func) (arg);

This is used by Thread::Fork and for interrupt handlers, as well as a couple of other places.
