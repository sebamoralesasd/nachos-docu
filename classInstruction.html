Detailed Description

See copyright.h for copyright notice and limitation of liability and disclaimer of warranty provisions. The following class defines an instruction, represented in both:

  • undecoded binary form;
  • decoded to identify:
    • operation to do;
    • registers to act on;
    • any immediate operand value.
