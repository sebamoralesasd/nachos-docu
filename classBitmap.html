Detailed Description

A data structure defining a bitmap – an array of bits each of which can be either on or off. It is also known as a bit array, bit set or bit vector.

The bitmap is represented as an array of unsigned integers, on which we do modulo arithmetic to find the bit we are interested in.

The data structure is parameterized with with the number of bits being managed.

Most useful for managing the allocation of the elements of an array – for instance, disk sectors, or main memory pages.

Each bit represents whether the corresponding sector or page is in use or free.
