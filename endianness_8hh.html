Routines for converting Words and Short Words to and from the simulated machine's format of little endian. If the host machine is little endian (DEC and Intel), these end up being NOPs.

What is stored in each format:

Host byte ordering User registers and kernel data structures. Simulated machine byte ordering Main memory.
