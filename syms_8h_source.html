196 /*
197  * This (part of the) file contains the definition of the Third Eye Symbol
198  * Table.
199  *
200  * Symbols are assumed to be in 'encounter order' - i.e. the order that
201  * the things they represent were encountered by the compiler/assembler/
202  * loader. EXCEPT for globals! These are assumed to be bunched together,
203  * probably right after the last 'normal' symbol. Globals ARE sorted
204  * in ascending order.
205  *
206  * -----------------------------------------------------------------
207  * A brief word about Third Eye naming/use conventions:
208  *
209  * All arrays and index's are 0 based.
210  * All "ifooMax" values are the highest legal value PLUS ONE. This makes
211  * them good for allocating arrays, etc. All checks are "ifoo < ifooMax".
212  *
213  * "isym" Index into the SYMbol table.
214  * "ipd" Index into the Procedure Descriptor array.
215  * "ifd" Index into the File Descriptor array.
216  * "iss" Index into String Space.
217  * "cb" Count of Bytes.
218  * "rgPd" array whose domain is "0..ipdMax-1" and RanGe is PDR.
219  * "rgFd" array whose domain is "0..ifdMax-1" and RanGe is FDR.
220  */
