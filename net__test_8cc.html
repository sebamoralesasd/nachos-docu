void MailTest ( int  farAddr)

Test out message delivery between two "Nachos" machines, using the Post Office to coordinate delivery.

Two caveats:

  1. Two copies of Nachos must be running, with machine ID's 0 and 1: ./nachos -m 0 -o 1 & ./nachos -m 1 -o 0 &
    1. You need an implementation of condition variables, which is not provided as part of the baseline threads implementation. The Post Office will not work without a correct implementation of condition variables.

See copyright.h for copyright notice and limitation of liability and disclaimer of warranty provisions. Test out message delivery, by doing the following:

  1. Send a message to the machine with ID "farAddr", at mail box #0.
  2. Wait for the other machine's message to arrive (in our mailbox #0).
  3. Send an acknowledgment for the other machine's message.
  4. Wait for an acknowledgement from the other machine to our original message.
